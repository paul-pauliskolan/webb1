Modul 2 · Kapitel 2.3

Formulär från grunden

Låt användaren skriva, välja och skicka information.

Mål

Efter kapitlet ska du kunna...

  • bygga ett formulär med form, label och input
  • förklara action, method, id, for och name
  • använda textarea, select, radio och checkbox
  • skicka formuläret med en submit-knapp

Formulärets ram

form samlar fälten

<form action="https://httpbin.org/post"
      method="post">
  ...
</form>
  • form omsluter fält som hör ihop.
  • action anger vart informationen skickas.
  • method anger hur informationen skickas.

Testtjänst

Vad är httpbin?

httpbin.org/post tar emot en testförfrågan.

Tjänsten visar tillbaka vilka formulärfält och värden som skickades.

Du kan därför öva utan att först bygga en egen server.

GET och POST

GETPOST

Label och input

Beskriv fältet tydligt

<label for="first-name">Förnamn:</label>
<input type="text"
       id="first-name"
       name="first-name"
       placeholder="Ada">

label berättar för användaren vad som ska skrivas.

Kopplingen

for måste matcha id

for="first-name"
på labelid="first-name"
på input

Samma värde kopplar etiketten till rätt fält.

När användaren klickar på etiketten aktiveras fältet.

name

Namnet på data som skickas

<input type="text" name="first-name" value="Ada">
Namn
first-nameVärde
Ada

Mottagaren får paret first-name=Ada.

Utan name skickas fältets värde normalt inte med.

placeholder

Ett exempel – inte en label

<input type="text" placeholder="Ada">

placeholder visas i fältet innan användaren skriver.

Texten försvinner när användaren börjar skriva.

Använd fortfarande en synlig label.

Textarea

Text på flera rader

<label for="message">Meddelande:</label>
<textarea id="message"
          name="message"
          rows="5"></textarea>

textarea passar längre svar och meddelanden.

Select

Välj ett alternativ i en meny

<label for="topic">Ämne:</label>
<select id="topic" name="topic">
  <option value="html">HTML</option>
  <option value="css">CSS</option>
</select>

select skapar menyn. Varje option är ett val.

Radio

Välj exakt ett alternativ

<label>
  <input type="radio" name="level" value="beginner">
  Nybörjare
</label>
<label>
  <input type="radio" name="level" value="advanced">
  Van
</label>

Samma name gör att radioknapparna tillhör samma grupp.

Nivå

Checkbox

Ett oberoende val

<label>
  <input type="checkbox"
         name="html"
         value="yes">
  Jag vill lära mig HTML
</label>

En kryssruta kan vara av eller på oberoende av andra val.

Radio eller checkbox?

Välj rätt kontroll

RadioCheckbox
Ett val ur en grupp.Ett eller flera oberoende val.
Exempel: nybörjare eller van.Exempel: HTML och CSS.

Knappar

submit skickar formuläret

<button type="submit">Skicka</button>

Försöker skicka formuläret.

<button type="button">Visa hjälp</button>

Skickar inte formuläret och kan senare kopplas till JavaScript.

Sammanfattning

Kom ihåg

  • form samlar fälten; action och method styr skickandet.
  • for och id kopplar label till rätt fält.
  • name anger namnet på data som skickas.
  • Radio används för ett val; checkbox för oberoende val.
  • button type="submit" skickar formuläret.
