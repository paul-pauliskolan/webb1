Media queries

Ändra CSS när ett villkor är uppfyllt.

Mål

Efter kapitlet ska du kunna...

  • förklara vad en media query gör

  • testa layouten i utvecklarverktygen

Villkorad CSS

Vad gör en media query?

@media (min-width: 48rem) {
  /* CSS som bara gäller från 48rem */
}

Reglerna inuti blocket används bara när villkoret är sant.

Grundregeln används

Mobile first

Börja med den lilla layouten

.layout {
  display: grid;
  gap: 1rem;
}

Grundreglerna ska fungera på liten skärm utan en media query.

Bygg vidare

Lägg till den större layouten

@media (min-width: 48rem) {
  .layout {
    grid-template-columns: 2fr 1fr;
  }
}

Från 48rem får layouten två kolumner.

Interaktiv Grid

En kolumn blir två

100vw är alltid hela viewportens bredd och är därför inte ett användbart värde på detta reglage.

Breakpoint

En punkt där layouten behöver ändras

  • Före brytpunkten fungerar den lilla layouten.
  • Efter brytpunkten används de nya reglerna.

Välj brytpunkt

Låt innehållet bestämma

  • Gör fönstret långsamt smalare.
  • Stanna när innehållet blir trångt eller går sönder.
  • Lägg brytpunkten där layouten behöver en ny form.

Utgå inte från en viss telefonmodell.

Vanlig ändring

Flexbox: kolumn till rad

.actions {
  display: flex;
  flex-direction: column;
}
@media (min-width: 40rem) {
  .actions { flex-direction: row; }
}

Prova själv

Bygg en mobile first-layout

  1. Skapa huvudkolumn och sidokolumn.
  2. Låt grundläget vara en kolumn.
  3. Lägg till två kolumner med min-width.
  4. Ändra fönsterbredden långsamt och justera brytpunkten.

Sammanfattning

Kom ihåg

  • Media queries kör CSS när ett villkor stämmer.
  • Mobile first börjar med den lilla layouten.
  • min-width lägger till regler från brytpunkten och uppåt.
  • Innehållets behov bestämmer brytpunkten.
  • Testa alla bredder mellan smalt och brett.
