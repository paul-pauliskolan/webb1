Hämta och ändra element

DOM gör att JavaScript kan arbeta med HTML-sidan.

Mål

Efter kapitlet ska du kunna...

  • hämta element med querySelector
  • ändra text med textContent
  • ändra klasser med classList
  • reagera på klick med addEventListener

Första träffen

querySelector hämtar ett element

const paragraph = document.querySelector("p");

Om flera p-element matchar returneras det första.

Senare finns metoder för att hämta flera element, men här arbetar vi med ett.

Variabel

Spara elementet med const

const button = document.querySelector(".theme-button");
const message = document.querySelector("#message");
  • button och message är namn vi väljer.
  • Variablerna gör elementen enkla att använda igen.

Interaktiv textContent

Byt texten

Event

Något händer i webbläsaren

  • click: användaren klickar.
  • input: ett fält ändras.
  • submit: ett formulär skickas.

I detta kapitel börjar vi med click.

addEventListener

Lyssna efter ett klick

button.addEventListener("click", function () {
  message.textContent = "Du klickade på knappen";
});

Funktionen körs inte direkt. Den väntar tills klicket sker.

Växla klassen is-active

Klassen är av.

Den här rutan ändras av CSS-klassen.

JavaScript + CSS

Låt språken samarbeta

/* CSS */
.box.is-active { background: lightgreen; }

// JavaScript
box.classList.toggle("is-active");

Undvik att lägga all design direkt i JavaScript. Växla hellre en tydligt namngiven klass.

Samlat exempel

Hitta, lyssna och ändra

const button = document.querySelector("#action"); const box = document.querySelector(".box"); button.addEventListener("click", function () { box.classList.toggle("is-active"); box.textContent = "Rutan ändrades"; });
Rutan väntar på ett klick.

Felsökning

Om elementet inte hittas

  1. Kontrollera att selektorn matchar HTML.
  2. Kom ihåg punkt för klass och # för id.
  3. Kontrollera stavning och stora/små bokstäver.
  4. Kontrollera att skriptet använder defer.
  5. Läs felmeddelandet i Console.

Prova själv

Bygg en enkel interaktion

  1. Skapa en knapp och ett stycke i HTML.
  2. Hämta båda med querySelector.
  3. Lyssna på knappens click.
  4. Ändra styckets textContent.
  5. Lägg till en klassväxling.

Sammanfattning

Kom ihåg

  • querySelector hittar ett element.
  • textContent läser eller ändrar text.
  • addEventListener reagerar på händelser.
  • classList.toggle växlar en klass.
  • JavaScript ändrar tillstånd – CSS visar utseendet.
