JavaScript på webbsidan

Vad du lär dig

  • Förstå vad JavaScript tillför till HTML och CSS.
  • Koppla en JavaScript-fil till sidan.
  • Hämta och ändra element i DOM.
  • Lyssna på events från knappar och formulär.
  • Bygga små interaktioner utan ramverk.
