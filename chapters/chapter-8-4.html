Kap 8.4 - Publicera med GitHub Pages

Publicering gör din webbplats tillgänglig på en riktig webbadress.

Mål

  • Förstå vad GitHub Pages används till.
  • Kontrollera filnamn och startsida före publicering.
  • Testa den publicerade sidan efteråt.

Vad GitHub Pages gör

GitHub Pages kan publicera statiska webbsidor direkt från ett repository. Det passar för HTML, CSS, JavaScript och bilder som inte kräver serverkod.

Före publicering

Kontrollera att startsidan heter index.html, att länkar fungerar, att bildsökvägar stämmer och att filnamn matchar exakt med stora och små bokstäver.

Efter publicering

Öppna den publicerade adressen och testa sidan igen. Problem kan synas först när webbplatsen ligger på GitHub Pages, särskilt om sökvägar till CSS, JavaScript eller bilder är fel.
