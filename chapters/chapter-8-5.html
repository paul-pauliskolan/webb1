Kap 8.5 - Slutprojekt: egen webbplats

Nu samlar du planering, HTML, CSS, JavaScript, testning och publicering i ett projekt.

Mål

  • Planera och bygga en egen mindre webbplats.
  • Använda semantisk HTML, responsiv CSS och enkel JavaScript där det passar.
  • Testa, förbättra och publicera projektet.

Projektets delar

Välj ett ämne med tydligt syfte och målgrupp. Webbplatsen ska ha genomtänkt struktur, fungerande navigation, läsbart innehåll, responsiv layout och bilder eller media som används ansvarsfullt.

Arbeta stegvis

  1. Planera syfte, målgrupp och innehåll.
  2. Skissa layout för mobil och desktop.
  3. Bygg HTML-strukturen.
  4. Styla med CSS och testa responsivitet.
  5. Lägg till enkel JavaScript om det förbättrar sidan.
  6. Testa och publicera.

Bedöm din egen sida

Kontrollera om sidan är begriplig, fungerar på mobil, har tydliga länkar, rimlig kontrast, fungerande formulär eller interaktioner och inga saknade filer i Network-panelen.
